# AGENT_TEAM_FLOW — Flow ทำงานละเอียด

## 0. Intake
Boss ส่งคำสั่ง ไฟล์ ลิงก์ ภาพ หรือเป้าหมาย

## 1. Captain Creates Job Card
- แปลงคำสั่งเป็น One Piece / Gold
- ระบุ output ที่ต้องส่ง
- กำหนด risk/approval gate

## 2. Navigator Routes
- เลือก Hat Set
- แตก MiroFish
- แบ่ง Mission

## 3. Archive Reads Sources
- ค้น Drive / uploaded files / GitHub / web
- ถอด SOP / data / rules
- ทำ evidence map

## 4. Team Builds
- Forge สร้างระบบหรือ prototype
- Copy เขียน content/report
- Sniper ทำ hook/growth
- Blade ตรวจความคม/SEO/logic

## 5. Doctor Validates
- ตรวจ test result
- ตรวจ evidence
- reject ถ้าไม่มีหลักฐาน

## 6. Jin Governance Gate
- ตรวจ cost/delete/public deploy/sensitive risk
- ถ้าเสี่ยงให้ส่ง Pending Boss Approval

## 7. Brook Summarizes
- สรุปไทย/อังกฤษ
- ทำ decision log
- บอก next action

## 8. Captain Closes
- ปิด Job เมื่อ receipt verified
- เพิ่ม XP/Level/Season
- อัปเดต LLM_WIKI หรือ SKILL.md ถ้าเจอ pattern ใหม่
