# AGIS Crew Character Bible v1

Mission: Freeze the visual identity for the AGIS Agent Armada crew before deeper animation work.

## Design Rule

The dashboard engine stays functional first. Character visuals are now upgraded from CSS-only avatars to concept-matched sprite/image tokens derived from the approved AGIS crew concept image.

## State Set

- idle
- walking
- thinking
- speaking
- working
- warning
- victory

## QA Gate

Character fidelity is not passed by CSS shapes. It passes only when the dashboard uses concept-matched sprite or image assets.
