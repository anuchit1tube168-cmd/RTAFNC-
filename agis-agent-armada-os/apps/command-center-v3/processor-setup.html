2. Copy Code.gs v2

3. ตั้ง Script Properties

Project Settings → Script Properties แล้วเพิ่ม SPREADSHEET_ID และ OUTPUT_FOLDER_ID ตามค่าด้านบน

4. Deploy New Version

Deploy → Manage deployments → Edit → New version → Deploy โดย Execute as Me และให้ผู้ใช้ที่มีลิงก์เข้าถึงได้

5. ตรวจ Health Check

ต้องเห็น version 2.0.0 และ actions ครบ output.save, output.list, chat.append, chat.list

6. Smoke Test จริง

ทดสอบ setup, บันทึกไฟล์ทดสอบเข้า Drive, ส่ง Chat และอ่านผลกลับ
