# LLM Wiki — evidence-gate
- **transcript vs outcome**: transcript = สิ่งที่ agent เล่า, outcome = สิ่งที่โลกเปลี่ยนจริง — gate เชื่ออย่างหลังเท่านั้น
- **ความสดของหลักฐาน**: screenshot เก่า/URL cache = near-miss ที่หลอก gate บ่อยสุด
- **สองชั้นสำหรับ high-risk**: หลักฐาน 2 ชนิดอิสระกัน โกหกพร้อมกันยาก
- **gate คือหัวใจของระบบ Fable**: skill อื่นทั้งหมดพึ่ง gate นี้ในการสะสมความน่าเชื่อถือ
