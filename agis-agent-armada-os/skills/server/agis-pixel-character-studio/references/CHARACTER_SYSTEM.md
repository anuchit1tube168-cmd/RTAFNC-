# AGIS Character System

## Visual Direction
- dark navy base
- gold role accent
- readable at 64×64
- original adventure-command-center crew
- same body proportions across the fleet
- accessory identifies role before text is read

## States
- idle
- focused
- working
- alert
- done
- level-up

## Role Accessories
- AGIS: crown
- Navigator: compass
- Archaeologist: book
- Shipwright: hammer
- Doctor: cross
- Swordsman: sword
- Cook: pan
- Sniper: target
- Clone: clone rings

## Server Rule
The server generates base characters and idle motion. Complex walking, attack, and deck-navigation animation belongs to Phase 2 PixiJS/Phaser tooling, not this endpoint.
