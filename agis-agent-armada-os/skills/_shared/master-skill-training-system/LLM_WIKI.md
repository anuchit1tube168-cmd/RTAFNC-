# LLM_WIKI — Master Skill Training System

## One Line
Turn every important capability into a reusable Skill Package with owner agent, test job, evidence, receipt, and level-up path.

## Core Pattern
Roadmap → Skill Package → Agent Injection → Real Test → Evidence → Receipt → Level Up

## P0 Skills
1. Requirement Extraction
2. SOP to Workflow
3. Google Drive Knowledge Index
4. Google Sheet State DB
5. Apps Script Processor
6. GitHub Repo Operating
7. Lovable UI Builder
8. Evidence Gate / Receipt
9. QA Debug
10. Agent Routing

## Rule
Do not move to P1/P2 before P0 skills can operate in real jobs with receipts.
