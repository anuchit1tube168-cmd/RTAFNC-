# AGENT_INJECTION — Master Skill Training System

## Purpose
ติดตั้งระบบฝึก Skill MD เข้า AGIS Crew เพื่อให้ทุก agent รู้ว่าต้องฝึกอะไร ใช้ทดสอบอย่างไร และเลเวลอัปเมื่อไร

## Crew Injection

| Agent | Skill Training Role |
|---|---|
| Captain AGIS | อนุมัติ priority P0/P1/P2 และตัดสินใจว่า skill ใดต้องสร้างก่อน |
| Navi Map | map skill กับ job type, hat set, mission split และ owner agent |
| Archive Eye | ถอดข้อมูลจาก SOP/Drive/YouTube/Prompt เป็น skill requirements |
| Forge Dev | สร้าง workflow, code, Apps Script, GitHub file, Lovable prompt จาก skill |
| Bug Doctor | สร้าง test plan และตรวจว่า skill ใช้ได้จริง |
| Jin Guard | ตรวจ risk, privacy, cost, public deploy และ tool-dependence |
| Brook Log | สรุป receipt, release note และ lesson learned |
| Copy Chef | เขียน skill package ให้อ่านง่าย ใช้ฝึก Boss และ agent ได้ |
| Hook Sniper | แปลง skill เป็น marketing/content/testing angle เมื่อเกี่ยวข้อง |
| Blade SEO | จัด taxonomy, naming, keyword, searchability ของ skill library |

## Training Loop for Every Agent
1. Learn skill concept
2. Read SKILL.md
3. Read LLM_WIKI.md
4. Follow CHECKLIST.md
5. Run TEST_PLAN.md
6. Use in real job
7. Create receipt
8. Update AgentSkillMap

## Level Rule
- Useful: เข้าใจและใช้ตาม checklist ได้
- Senior Candidate: ใช้ในงานจริงพร้อม receipt ได้
- Master Candidate: ใช้ซ้ำได้ 3+ job types
- Clone Ready: clone agent ใช้ skill ได้โดยไม่พัง

## P0 First
ทุก agent ต้องติดตั้ง P0 core ก่อนขยายไป P1/P2
