# Context Hat Switching — SKILL.md

## Purpose
ใช้เมื่อ Job เปลี่ยนบริบท เช่น ตัดต่อคลิป ขายของ SEO เว็บแอป รายงานราชการ Shopee หรือวิเคราะห์ข้อมูล ให้ agent ทุกตัวเปลี่ยนหมวกตามงานได้ทันที

## Inputs
- Boss Command
- One Piece / Gold
- Job Type
- Source files / links / screenshots / Drive files
- Constraint / deadline / risk

## Process
1. Classify Job Type
2. Select Hat Set
3. Assign primary and support agents
4. Activate matching Skill MD
5. Create missions
6. Set evidence requirement
7. Run QA and approval gate
8. Update LLM_WIKI when new reusable pattern is found

## Hat Sets
- Video Editing
- Sales Marketing
- SEO
- WebApp GitHub Lovable
- Google Drive Big Data
- Report SOP Thai DOCX
- Shopee Affiliate
- QA Risk Governance

## Output Contract
- Hat Set selected
- Agent assignment
- Skill MD list
- Mission list
- Evidence list
- Receipt JSON
