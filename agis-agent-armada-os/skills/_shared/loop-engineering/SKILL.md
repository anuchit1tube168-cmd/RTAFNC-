# Loop Engineering — SKILL.md

## Purpose
มาตรฐานกระบวนการสร้างงานของ AGIS Agent Armada OS ใช้กับทุก Job ทุกบริบท

## Core Loop
Intent → Spec → Micro-Phase → Plan → Work → Guard → Validate → Review → Revise → Report & Receipt

## Decision Loops
- SWOT: ใช้ก่อนเริ่มงานใหญ่
- OODA: ใช้เมื่อต้องตัดสินใจเร็ว
- PDCA: ใช้เมื่อต้องปรับปรุงต่อเนื่อง
- MiroFish: ใช้แตกปัญหาเป็น cause/skill/agent/mission/evidence

## Process
1. Intake Boss Command
2. Create Job Card
3. Define One Piece / Gold
4. Select Hat Set
5. Run SWOT/OODA/PDCA as needed
6. Split micro-phases
7. Assign agents
8. Work with evidence
9. Validate
10. Revise
11. Store receipt
12. Level up and update wiki/skill

## Output Contract
- Job Card
- Mission Board
- Agent Assignment
- Evidence Checklist
- Receipt JSON
- TH/EN Summary
