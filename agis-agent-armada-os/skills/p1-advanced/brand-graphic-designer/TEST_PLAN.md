# TEST_PLAN — Brand Graphic Designer Skill

## Test 1: Raw Topic to Image Concept
Input: one rough topic
Expected output:
- Headline
- Body
- Accent note
- Visual direction
- Image prompt
- QA result

Pass if:
- One idea only
- Text is short
- No invented facts

## Test 2: Brand Consistency
Input: brand colors and personality
Expected:
- Prompt uses brand tone and colors
- Forbidden colors are avoided
- Visual style is consistent

Pass if:
- No brand drift
- Suitable for the chosen platform

## Test 3: Long Product Info Compression
Input: long product details
Expected:
- Meaning preserved
- Copy shortened
- Clear headline

Pass if:
- No important meaning changed
- No unsupported claim added

## Test 4: Placeholder Rule
Input: request that needs real product/person photo
Expected:
- Prompt uses placeholder only
- Does not invent real product image

## Test 5: Crew Handoff
Input: one campaign idea
Expected:
- Copy Chef creates layout brief
- Hook Sniper improves headline/CTA
- Jin Guard verifies claims
- Blade SEO checks platform keyword
- Forge Dev prepares Canva/Lovable handoff if needed

## Evidence
- Final image concept
- Final prompt
- QA checklist result
- Generated image/design link if available
- Receipt
