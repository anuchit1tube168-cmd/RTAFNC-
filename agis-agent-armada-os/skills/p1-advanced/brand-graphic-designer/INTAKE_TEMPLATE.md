# INTAKE_TEMPLATE — Brand Graphic Designer

## Boss Input Template
ใช้แบบฟอร์มนี้เวลา Boss ต้องการให้ลูกเรือสร้างภาพคอนเทนต์ 1 ภาพ

```text
แบรนด์:
ธุรกิจ/สินค้า/บริการ:
กลุ่มเป้าหมาย:
บุคลิกแบรนด์:
โทนสีหลัก:
สีที่ห้ามใช้:
สไตล์ภาพ:
แพลตฟอร์ม: Instagram / Facebook / TikTok / Ads / Carousel
หัวข้อภาพ:
สิ่งที่ต้องมี:
สิ่งที่ห้ามทำ:
CTA ถ้ามี:
```

## Required Output
```text
Headline:
Body:
Accent note:
Visual direction:
Image generation prompt:
Design QA result:
```

## Minimum Input Rule
ถ้า Boss ให้ข้อมูลไม่ครบ ให้ใช้เฉพาะข้อมูลที่มี และอย่าเพิ่มข้อเท็จจริงเอง

## Missing Info Handling
ถ้าขาดข้อมูล ให้ถามเฉพาะข้อมูลจำเป็นที่สุด 1-3 ข้อ หรือสร้างด้วย placeholder ถ้าไม่กระทบความจริง

## QA Gate
ก่อนส่งออก ต้องเช็ก:
- ภาพไม่รก
- สีไม่หลุดแบรนด์
- ข้อความอ่านง่าย
- Headline เด่นสุด
- ไม่เพิ่มข้อมูลที่ไม่ได้ให้มา
- เหมาะกับ platform
