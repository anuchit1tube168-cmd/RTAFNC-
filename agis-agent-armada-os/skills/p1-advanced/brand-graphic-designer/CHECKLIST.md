# CHECKLIST — Brand Graphic Designer Skill

## Brand Intake
- [ ] Brand name is known
- [ ] Product/service is known
- [ ] Target audience is known
- [ ] Brand personality is known
- [ ] Main colors are known
- [ ] Forbidden colors are known or marked none
- [ ] Required elements are known
- [ ] Prohibited elements are known
- [ ] Platform is selected

## One Image Rule
- [ ] One image only
- [ ] One core idea only
- [ ] Headline is the main focal point
- [ ] Body copy is short
- [ ] No crowded text

## Visual QA
- [ ] Image is not cluttered
- [ ] Color does not drift from brand
- [ ] Text readable on mobile
- [ ] Visual hierarchy is clear
- [ ] Main focal point is obvious
- [ ] Style matches prior brand assets
- [ ] Placeholder is used if real photo is required

## Compliance QA
- [ ] No unsupported claims
- [ ] No fake promotion added
- [ ] No invented price/guarantee/result
- [ ] Required logo/username/site/phone included if provided
- [ ] Avoids direct-sale look when prohibited

## Done
- [ ] Headline created
- [ ] Body created
- [ ] Accent note created
- [ ] Visual direction created
- [ ] Image generation prompt created
- [ ] QA result created
- [ ] Evidence/receipt created after use
