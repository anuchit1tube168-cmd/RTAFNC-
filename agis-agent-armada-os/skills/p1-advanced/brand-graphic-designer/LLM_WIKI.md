# LLM_WIKI — Brand Graphic Designer Skill

## One Line
Turn a rough content idea into one clean, brand-safe image concept and image-generation prompt for social media.

## Use When
- Boss gives a product/topic/content idea
- Need Instagram/Facebook/TikTok/Ads/Carousel image concept
- Need brand-consistent visual prompt
- Need QA before generating or sending to designer

## Output Format
- Headline
- Body
- Accent note
- Visual direction
- Image generation prompt
- Design QA result

## Main Agent
Copy Chef

## Support Agents
- Hook Sniper: hook and CTA
- Blade SEO: keyword/searchability/platform framing
- Jin Guard: claims and brand compliance
- Forge Dev: Canva/Lovable/UI build handoff

## Rules
- One image only
- One idea only
- Short copy
- Headline strongest
- No unsupported claims
- Placeholder only when real photo is needed
- Keep brand style consistent

## Done
Done only when concept + prompt + QA pass + evidence/receipt exist.
