# LLM Wiki — risk-compliance
- **data minimization**: ระบบเก็บ/แสดงเฉพาะที่จำเป็นต่องาน — dashboard ครูไม่จำเป็นต้องโชว์เลขประชาชน
- **audit log = เกราะของคนทำงาน**: พิสูจน์ว่าตรวจแล้วตามขั้นตอน เมื่อถูกตั้งคำถามภายหลัง
- **เสนอทางแก้เสมอ**: compliance ที่ปัดตกอย่างเดียวจะถูก bypass — ต้องเป็นด่านที่ช่วยให้งานผ่านแบบปลอดภัย
