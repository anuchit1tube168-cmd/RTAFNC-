## Core Insight
AI/Claude Code/GPT Project จะเก่งขึ้นเมื่อมี 5 ชั้นนี้:
1. Context — รู้โปรเจกต์ เป้าหมาย กติกา ไฟล์ และข้อจำกัด
2. Prompt — สั่งงานแบบมี role, output, guard, acceptance criteria
3. Automation — มี workflow ซ้ำได้ ไม่ทำมือทุกครั้ง
4. Debug — ตรวจ error, test, fix ทีละจุด
5. Delivery — ส่งงานพร้อม evidence, receipt, deploy/test note
