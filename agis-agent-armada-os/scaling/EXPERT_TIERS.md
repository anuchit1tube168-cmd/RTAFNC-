# EXPERT_TIERS — Specialist / Warlord / Emperor

## Purpose
กำหนดระดับผู้เชี่ยวชาญที่สามารถเติมเข้ากองเรือเมื่อ Core Crew หรือ Clone Crew ไม่พอ

## Tier 1: Specialist Crew
ใช้เมื่อมีงานเฉพาะทางแต่ยังเป็น low/medium risk

ตัวอย่าง:
- Data Engineer: database, schema, ETL, Google Sheet/Supabase
- Apps Script Engineer: GAS, Web App, triggers
- SEO Analyst: keyword, technical SEO, Search Console
- Video Editor: storyboard, edit plan, subtitle, render QA
- UX Designer: flow, wireframe, responsive UI
- Prompt Engineer: prompt pack, agent instruction
- Document Officer: DOCX/PDF/Thai official writing
- Shopee Affiliate Analyst: product rank, hook, warning

## Tier 2: Warlord Expert
ใช้เมื่อ blocker ยากและต้องการ expert ที่ core crew ทำแทนไม่ได้

ตัวอย่าง:
- Cybersecurity Guard
- Legal/Policy Reviewer
- Financial Risk Reviewer
- Medical/Nursing Domain Reviewer
- Cloud Architect
- Data Governance Expert
- Advanced Automation Architect
- Brand/Creative Director

## Tier 3: Emperor Board
ใช้เมื่อเป็นงานระดับองค์กร ยุทธศาสตร์ หรือเปลี่ยนระบบใหญ่

สมาชิกที่ควรมี:
- Strategy Emperor: ตัดสินใจภาพใหญ่
- Risk Emperor: ความเสี่ยง/กฎหมาย/ข้อมูล
- Tech Emperor: architecture/deploy/security
- Data Emperor: schema/big data/reporting
- Growth Emperor: market/SEO/content/affiliate

## Who Can Recruit
- Captain เสนอ recruit
- Navi ระบุ gap
- Jin Guard ตรวจ risk
- Boss อนุมัติเมื่อเป็น Warlord หรือ Emperor

## Recruitment Evidence
ต้องมีเหตุผลว่า:
- blocker คืออะไร
- core crew ทำไมไม่พอ
- ต้องการ expertise อะไร
- expert จะส่ง output อะไร
- ใช้แล้วจะ update skill md อะไรกลับเข้าระบบ
