# 3D_PIXEL_CREW_SPEC — Original Crew Characters

## Purpose
กำหนดตัวละคร 3D pixel ของ AGIS Agent Armada OS ให้มีชีวิตบนเรือ ขยับได้ พูดคุยได้ และเปลี่ยนบทบาทตาม Job โดยเป็น original IP-safe characters

## Art Style
- 3D pixel / voxel style
- รายละเอียดสูงแต่ยังเป็น blocky game-like
- Chibi-proportion หรือ stylized professional crew
- Animation loop สั้น: idle, walk, talk, work, celebrate, warning
- Camera: isometric ship deck / command room / island scene

## Core Crew Visuals

### Captain AGIS
- Role: Orchestrator
- Visual: red-gold captain coat, command wheel, compass badge
- Animation: points to map, raises flag, approves gate

### Navi Map
- Role: Router
- Visual: blue navigator jacket, glowing map tablet
- Animation: walks to map table, draws route

### Archive Eye
- Role: Knowledge Analyst
- Visual: dark scholar coat, archive satchel, knowledge stone
- Animation: opens scrolls, scans documents

### Forge Dev
- Role: Builder
- Visual: mechanic apron, tools, glowing blueprint
- Animation: hammers interface, deploy lever

### Bug Doctor
- Role: Validator
- Visual: medic/engineer hybrid, QA scanner
- Animation: scans output, flags bug, stamps pass

### Blade SEO
- Role: Precision Guard
- Visual: disciplined guard, sharp pointer, checklist blade
- Animation: cuts duplicate cards, highlights issues

### Copy Chef
- Role: Content Writer
- Visual: cook-writer hybrid, notebook, subtitle ladle

### Hook Sniper
- Role: Growth Agent
- Visual: lookout/sniper, telescope, signal flags
- Animation: spots market, fires hook signal

### Brook Log
- Role: Story/Log
- Visual: bard/logkeeper, ledger, tiny instrument
- Animation: writes log, narrates bilingual summary

### Jin Guard
- Role: Governance
- Visual: helmsman guardian, anchor shield, policy seal
- Animation: blocks risky gate, stamps approval

## Interaction Rules
- Each agent must show currentHat badge
- Each agent must show status: standby, working, review, blocked, done
- Dialogue bubble supports Thai and English
- Clicking agent opens Skill MD / current mission / evidence
- When clone created, ghost/mini version appears with clone label
- When level up, agent gets aura and new badge

## Do Not
- Do not copy faces, costumes, names, ship designs, logos, or scenes from copyrighted anime
- Do not use exact crew names from existing series in production assets

## Goal
ให้ความรู้สึกเหมือนกองเรือผจญภัยที่มีชีวิต แต่เป็นจักรวาล AGIS Original ใช้จริงในองค์กรได้
