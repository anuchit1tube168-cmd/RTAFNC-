## Training Loop
1. Collect current context and identify blockers
2. Map blockers to missing skills
3. Prioritize skills as P0 / P1 / P2
4. Assign skill owner agent and backup agent
5. Define real-world test job
6. Create Skill Package files
7. Apply to real work
8. Collect evidence and receipt
9. Review after application
10. Update AgentSkillMap and level up

## Done Criteria
A skill is installed only when it has:
- Skill Package files
- Owner agent
- Real-world test task
- Evidence rule
- Receipt rule
- Agent injection record

## Immediate Priority
Install P0 Core Skill Pack first. Do not expand into P1/P2 before P0 can produce real evidence and receipts.
