# Lovable Prompt — 3D Pixel Fleet UI Layer

ใช้ prompt นี้เป็นขั้น visual/final layer หลัง P0 Processor ใช้งานได้แล้ว

## Prompt

สร้าง AGIS Agent Armada Command Center แบบ 3D pixel / voxel adventure fleet dashboard โดยเป็น original characters และ original world ไม่ใช้ชื่อ/ภาพ/เครื่องแต่งกาย/ฉากที่เหมือน copyrighted anime แบบเป๊ะ

### Main Scene
- Isometric 3D pixel ship deck
- Command wheel, map table, archive shelf, forge station, QA checkpoint, approval gate, logbook corner
- Agents move between stations based on current Job/Mission status

### Required Characters
Use original AGIS crew:
- Captain AGIS: Orchestrator
- Navi Map: Router
- Archive Eye: Knowledge Analyst
- Forge Dev: Builder
- Bug Doctor: Validator
- Blade SEO: Precision Guard
- Copy Chef: Content Writer
- Hook Sniper: Growth Agent
- Brook Log: Story/Log
- Jin Guard: Governance

### Animation States
- idle
- walkToStation
- working
- speaking
- blocked
- review
- approved
- levelUp
- cloneSpawn
- missionDone

### Scene Switching
Switch scene by jobType:
- Build/WebApp → Shipyard Island
- Research/SOP → Ancient Library Island
- Data/Sheet → Data Reef Island
- Video/Content → Studio Deck Island
- Marketing/SEO/Shopee → Market Port Island
- QA/Audit → Storm Checkpoint
- Governance/Risk → Council Harbor
- Scale/Expert → Alliance Gate

### Dialogue
Each agent has bilingual Thai/English speech bubble.
Examples:
- Captain: "เป้าหมายงานนี้คืออะไร?" / "What is the One Goal?"
- Navi: "กำลังแจกภารกิจ" / "Routing missions now."
- Bug Doctor: "ยังไม่มี evidence ปิดงานไม่ได้" / "Cannot close without evidence."
- Jin Guard: "ต้องขอ Boss approval" / "Boss approval required."

### UI Panels
- Job Board
- Mission Board
- Agent Status
- Evidence Gate
- Receipt Center
- Level/Clone Center
- Fleet Scale Center

### Safety
Production UI must use original characters and original scenes. It can keep adventure fleet energy but must not replicate existing copyrighted anime characters or scenes.
