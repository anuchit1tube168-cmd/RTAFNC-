# LLM_WIKI — AGIS Agent Armada OS

## Purpose
ให้ LLM/AI Agent เข้าใจระบบนี้อย่างเร็ว และทำงานต่อได้โดยไม่ต้องอ่านแชทยาวทั้งหมด

## Main Concept
- Gold / Goal / One Piece = Job เป้าหมายหลักจาก Boss ในแต่ละครั้ง
- Job = งานใหญ่ 1 รอบ
- Mission = งานย่อยที่แตกจาก Job
- Agent = บทบาทเฉพาะทาง
- Bounty = Difficulty + XP
- Haki = Skill Level
- Poneglyph = SOP / Rule / Knowledge ที่ซ่อนในเอกสาร
- Treasure Chest = Evidence / Receipt

## Mandatory Loop
Boss Command → Job Card → SWOT → OODA → MiroFish → Micro-Phase → Assign → Work → Guard → Validate → Review → Revise → Receipt → Level Up

## Decision Policy
Agent ตัดสินใจเองได้เมื่อ reversible, low-risk, internal-only
ต้องขอ Boss approval เมื่อ cost, delete data, public deploy, external send, sensitive decision, legal/medical/financial

## Retrieval Priority
1. Current uploaded files
2. Google Drive / project files / big data
3. GitHub repo/issues/PRs
4. Lovable project/prototype
5. Web search for current public info
6. Shopee only for shopping/affiliate product analysis
