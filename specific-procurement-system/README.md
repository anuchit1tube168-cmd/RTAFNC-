# SPS วพอ. · ระบบจัดซื้อจัดจ้าง วิธีเฉพาะเจาะจง

Static Single Page Application สำหรับ GitHub Pages สร้างใหม่จากภาพตัวอย่างและโครงฟีเจอร์ของระบบจัดซื้อจัดจ้างวิธีเฉพาะเจาะจง โดยวางเป็นโปรเจกต์แยกใน repo เดิม ไม่ทับงาน RTAFNC หลัก

## ลิงก์ใช้งาน

## ฟังก์ชันที่ทำแล้ว

- Login + Demo role quick login
- Dashboard สรุปจำนวนเอกสาร มูลค่ารวม และสถานะงาน
- รายการใบขอซื้อ/ขอจ้างแบบ Grid/List
- ค้นหา กรองประเภท และกรองสถานะ
- สร้างเอกสารใหม่พร้อมรายการพัสดุหลายรายการ
- Workflow สถานะ: ฉบับร่าง → เสนอตรวจสอบ → ผอ.เห็นชอบรายงาน → เสนอผลพิจารณา → อนุมัติสั่งซื้อ/จ้าง → ตรวจรับแล้ว → เสร็จสมบูรณ์
- หน้ารายละเอียดเอกสาร
- พิมพ์/บันทึก PDF เอกสารราชการ เช่น รายงานขอซื้อ/จ้าง รายละเอียดแนบ ใบเสนอราคา ใบสั่งซื้อ/จ้าง ใบตรวจรับ และรายงานสรุป
- Export CSV
- ผู้ขาย/ผู้รับจ้าง
- Audit log
- ตั้งค่าหน่วยงาน
- เก็บข้อมูลใน LocalStorage
- Responsive สำหรับมือถือ แท็บเล็ต และคอมพิวเตอร์

## หมายเหตุ

เวอร์ชันนี้เป็น Frontend Prototype แบบ Static สำหรับ GitHub Pages ข้อมูลเก็บในเครื่องผู้ใช้ผ่าน LocalStorage เหมาะสำหรับทดลอง Flow/UX และนำเสนอ ก่อนต่อยอดเป็นระบบจริงด้วย Google Apps Script + Google Sheets, Firebase, Supabase หรือ Backend API
