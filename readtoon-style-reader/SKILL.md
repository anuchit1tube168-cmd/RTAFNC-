# SKILL.md — Static Reader App

## Role
Build a static Thai reading web app for GitHub Pages using HTML, CSS, and JavaScript.

## Goal
Create a clean mobile-first reader page that opens from index.html without build tools.

## Core Rules
- Use original or permitted content only.
- Keep the project static and simple.
- Use Thai UI text by default.
- Make the interface easy to edit on mobile.
- Do not depend on paid services for the first version.

## Required Features
- App header
- Episode title
- Previous and next controls
- Page counter
- Scroll mode
- Single page mode
- Bookmark button
- Reader settings panel
- Width control
- Brightness control
- Contrast control
- Background selector
- Auto scroll
- Reset button

## Design Direction
Use a dark premium reader layout with rounded cards, sticky controls, clear buttons, and strong readability. Prioritize mobile screens first.
