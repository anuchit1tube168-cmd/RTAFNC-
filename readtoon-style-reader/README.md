# RTAFNC Toon Reader

Static webtoon reader demo for GitHub Pages.

## Features

- Mobile-first reading page
- Episode title and navigation
- Scroll mode and single-page mode
- Reader settings panel
- Width, brightness, contrast, background and auto-scroll controls
- Demo pages only by default
- Designed for original or licensed content only
