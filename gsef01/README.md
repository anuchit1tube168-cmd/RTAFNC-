เป้าหมายของโปรเจกต์นี้คือทำระบบขอทุนที่มี 2 ส่วนในหน้าเดียว:

1. แบบฟอร์มกรอกข้อมูลง่ายบนมือถือ
2. Official Preview แบบกระดาษ A4 แนวนอน อ้างอิงแบบ นร./กสศ.01 หน้า 1

สิ่งที่ทำแล้ว:

- โครงแบบฟอร์มราชการ A4 landscape
- โลโก้ กสศ. placeholder
- ช่องเลขบัตรประชาชน 13 ช่อง
- checkbox/radio พร้อมเครื่องหมายถูกสีน้ำเงิน
- ตารางสมาชิกครัวเรือนแบบหลายชั้น
- เพิ่ม/ลบสมาชิกครัวเรือน
- คำนวณรายได้รวมและรายได้เฉลี่ย
- อัปโหลดรูปประกอบและรูปบ้าน
- Print / Save PDF ผ่าน browser

หมายเหตุ: หน้า preview นี้ยังเป็น front-end prototype ส่วนการบันทึกข้อมูลจริงควรเชื่อม Google Apps Script/Google Sheets/Drive และ LINE OA ในขั้นถัดไป
