# Status Flow ระบบขอทุน

ลำดับสถานะหลัก:

```text
ร่าง -> รออาจารย์ที่ปรึกษา -> ผ่านอาจารย์ที่ปรึกษา -> อนุมัติ/ไม่อนุมัติ -> จ่ายเงินแล้ว
```

ทางย้อนกลับ:

```text
รออาจารย์ที่ปรึกษา -> ส่งกลับแก้ไข -> รออาจารย์ที่ปรึกษา
```

กฎใช้งาน:

- คำขอที่ยังเป็นร่างยังไม่เข้าสู่การพิจารณา
- ทุกการเปลี่ยนสถานะต้องมี log
- สถานะอนุมัติยังไม่เท่ากับจ่ายเงินแล้ว
