# RTAFNC Teaching Supervision Skill Package

ชุดสมองกลางสำหรับโครงการระบบนิเทศการสอน วพอ.พอ.

## วิธีใช้

1. วางโฟลเดอร์นี้ไว้ที่ root ของ project หรือคลัง Skill กลาง
2. ให้ Agent อ่าน `SKILL.md` และ `AGENTS.md` ก่อนทำงาน
3. เปิดด้วยคำสั่ง `rtafnc supervision mode` หรือ `ระบบนิเทศการสอน วพอ`
4. เมื่อแก้ระบบ ให้ตรวจ `CHECKLIST.md` และ `TEST_PLAN.md`
5. เมื่อ workflow เปลี่ยน ให้อัปเดต `SOP_MAP.md`, SOP และ `CHANGELOG.md`

## Files

- `SKILL.md` — คำสั่งหลัก
- `AGENTS.md` — กติกาสำหรับ coding agents
- `PROJECT_CONTEXT.md` — สถานะและแหล่งข้อมูล
- `SOP_MAP.md` — ผูก SOP กับระบบและการทดสอบ
- `DATA_DICTIONARY.md` — schema และกติกาข้อมูล
- `CHECKLIST.md` — release gate
- `TEST_PLAN.md` — แผนทดสอบและหลักฐาน
- `EXAMPLES.md` — ตัวอย่าง prompt
- `AGENT_INJECTION.md` — บล็อกฉีดให้ Agent
- `RECEIPT_TEMPLATE.md` — ใบรับรองผลการรัน/ส่งมอบ
- `LLM_WIKI.md` — หน้าอ้างอิงเร็ว
- `LOVABLE_KNOWLEDGE.md` — พร้อมวางใน project knowledge
- `CHANGELOG.md` — ประวัติรุ่น
