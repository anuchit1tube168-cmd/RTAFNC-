# Skill: LIFF Shell

## หน้าที่
จัดการหน้าเว็บที่เปิดภายใน LINE ให้เหมาะกับมือถือและส่งต่อข้อมูลการยืนยันตัวตนไปยังระบบกลาง

## ขั้นตอนหลัก
1. เริ่ม LIFF SDK
2. ตรวจสภาพแวดล้อมว่าเปิดใน LINE หรือ Browser
3. เรียกหน้าลงชื่อเข้าใช้เมื่อจำเป็น
4. รับข้อมูลเซสชันและส่งต่อไปยังสกิลยืนยันตัวตน
5. แสดงสถานะ DEMO, UNLINKED หรือ MEMBER
6. รองรับ Loading, Timeout, Offline และ Back navigation

## เกณฑ์สำเร็จ
- เปิดจาก Rich Menu ได้
- กลับเข้าสู่หน้าหลักหลังลงชื่อเข้าใช้
- หน้าไม่ค้างเมื่อเครือข่ายช้า
- ความดีและโรงพยาบาลแยกเมนูชัดเจน
