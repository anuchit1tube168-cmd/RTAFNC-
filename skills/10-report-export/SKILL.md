# Skill: Report Export

## หน้าที่
สร้างรายงานจากข้อมูลใน Supabase ไปยัง Google Sheets Template และส่งออกเป็น PDF สำหรับนักเรียนหรือเจ้าหน้าที่

## Workflow
1. รับ student UUID, report type และช่วงเวลา
2. ตรวจสิทธิ์ผู้ขอ
3. ดึงข้อมูลจากฐานข้อมูลตาม Scope
4. เติมข้อมูลลง Google Sheets Template
5. สร้าง PDF
6. บันทึกไฟล์กลับเข้า Private Storage
7. สร้าง document metadata
8. แจ้งนักเรียนเมื่อไฟล์พร้อม

## Report Types
- สรุปชั่วโมงความดีรายบุคคล
- รายงานความดีตามรุ่นหรือชั้นปี
- ทะเบียนไปโรงพยาบาลแบบจำกัดสิทธิ์
- ใบรับรองกิจกรรม
- เอกสารสรุปประจำปี

## Rules
- Google Sheets ไม่ใช่ฐานข้อมูลหลัก
- ไฟล์ใน Drive ต้อง Restricted
- รายงานสุขภาพต้องใช้สิทธิ์เฉพาะ
- ชื่อไฟล์ผู้ใช้เห็นใช้รหัสนักเรียนและปี แต่ระบบภายในใช้ UUID
