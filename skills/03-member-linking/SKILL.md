# Skill: Member Linking

## หน้าที่
ผูกบัญชี LINE ที่ผ่านการตรวจแล้วกับทะเบียนนักเรียนโดยใช้รหัสเปิดใช้งานครั้งเดียว

## ขั้นตอน
1. รับ verified LINE user ID
2. ตรวจว่าบัญชีนี้เคยผูกสมาชิกแล้วหรือไม่
3. ถ้ายังไม่ผูก ให้ขอ Activation Code
4. ตรวจ hash, วันหมดอายุ และสถานะใช้งาน
5. ตรวจว่านักเรียนหนึ่งคนไม่ถูกผูกกับหลายบัญชีโดยไม่ได้รับอนุญาต
6. บันทึก line account, auth user และ student UUID
7. ปิด Activation Code ทันทีหลังใช้
8. บันทึก Audit Log

## กรณีพิเศษ
- เปลี่ยนบัญชี LINE ต้องให้เจ้าหน้าที่ถอนการเชื่อมเดิมก่อน
- สมาชิก inactive หรือ suspended ห้ามเข้าสู่ข้อมูลส่วนตัว

## Output
- student UUID
- student code
- รุ่น ชั้นปี ปีการศึกษา
- member status
