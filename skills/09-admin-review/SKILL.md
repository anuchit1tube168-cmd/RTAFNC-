# Skill: Admin Review

## หน้าที่
จัดการคิวตรวจสอบของเจ้าหน้าที่โดยแยกสิทธิ์ตามโมดูลและบทบาท

## Queue Types
- ความดีรออนุมัติ
- เอกสารรอตรวจ
- ข่าวรอเผยแพร่
- ข้อมูลโรงพยาบาลสำหรับเจ้าหน้าที่แพทย์เท่านั้น

## Workflow ความดี
1. เปิดรายละเอียดและหลักฐาน
2. ตรวจประเภทกิจกรรมและจำนวนชั่วโมง
3. อนุมัติ ส่งกลับแก้ไข หรือไม่อนุมัติ
4. ระบุหมายเหตุ
5. บันทึกผู้ตรวจและเวลา
6. ส่งเหตุการณ์ไป Notification Hub

## Workflow โรงพยาบาล
- แสดงเฉพาะ Medical Role
- การเปิดข้อมูลทุกครั้งสร้าง Audit Log
- การแก้สถานะ follow_up หรือ resolved ต้องมีผู้ดำเนินการ

## Guardrails
- ผู้ดูแลระบบทั่วไปไม่เห็นข้อมูลสุขภาพโดยอัตโนมัติ
- ห้ามแก้ประวัติเดิมแบบไร้ร่องรอย
- Dashboard ผู้บริหารใช้ข้อมูลสรุปเท่านั้น
