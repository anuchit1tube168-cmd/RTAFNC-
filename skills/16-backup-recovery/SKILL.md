# Skill: Backup and Recovery

## หน้าที่
สำรองและกู้คืนฐานข้อมูล ไฟล์ และค่าคอนฟิกโดยลดผลกระทบต่อผู้ใช้

## ขอบเขต
- PostgreSQL database
- Private Storage metadata และไฟล์
- Migration history
- Server configuration ที่ไม่ใช่ข้อมูลลับ
- รายงานและเอกสารที่สร้างแล้ว

## แผนสำรอง
- สำรองฐานข้อมูลตามรอบเวลา
- ตรวจความสมบูรณ์ของไฟล์สำรอง
- แยกที่เก็บสำรองออกจากระบบหลัก
- กำหนดระยะเวลาเก็บตามประเภทข้อมูล

## Recovery Drill
1. สร้างสภาพแวดล้อมทดสอบ
2. กู้คืนฐานข้อมูล
3. ตรวจจำนวนแถวและความสัมพันธ์
4. ตรวจไฟล์ตัวอย่างจากทุก Bucket
5. ทดสอบ Login, RLS, ความดี และโรงพยาบาล
6. บันทึกเวลาในการกู้คืนและข้อผิดพลาด

## Rules
- ห้ามทดสอบกู้คืนทับ Production
- ไฟล์สำรองต้องเข้ารหัสและจำกัดสิทธิ์
- การลบหรือกู้คืนข้อมูลต้องมีผู้อนุมัติและ Audit Log
