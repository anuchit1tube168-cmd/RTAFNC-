# SOP IT Operations Skill

Skill สำหรับสร้าง/ปรับ SOP ฝ่ายไอที โดยยึดแนวทางจาก `skills-main` คือสั้น ใช้งานซ้ำได้ มีขั้นตอนชัด มีเกณฑ์จบงาน และใช้ progressive disclosure เท่าที่จำเป็น

## Files

- `SKILL.md` - skill หลัก พร้อม workflow, procedure catalog, terms, KPI และ template

## Install idea

คัดลอกโฟลเดอร์นี้ไปไว้ใน skills directory ของ agent ที่ใช้ แล้วเรียกใช้เมื่อจะทำ SOP เช่น:

- `/sop-it-operations สร้าง SOP งาน System Admin`
- `/sop-it-operations แปลงคู่มือแจ้งซ่อมไอทีเป็น SOP ที่ใช้งานจริง`
- `/sop-it-operations ตรวจ SOP นี้ว่าขาด evidence, KPI, rollback ตรงไหน`

## Scope

ใช้กับงานต่อไปนี้:

- System Admin
- IT Support Ticketing
- Incident / Alert / Monitoring
- Access Control / RBAC / MFA
- Backup / Restore / RPO / RTO
- Patch / Maintenance / Change Management
- SLA / KPI / Evidence / Audit log
