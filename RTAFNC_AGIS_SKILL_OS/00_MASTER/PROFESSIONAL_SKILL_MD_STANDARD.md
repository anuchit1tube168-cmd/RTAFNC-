# Professional Skill MD Standard — RTAFNC Edition

## Purpose
Turn every useful knowledge asset into a reusable Skill MD that can be read quickly, verified, and handed off to another AI agent or staff member.

## Required Sections
1. Name
2. Mission
3. Use When
4. Inputs
5. Outputs
6. Workflow
7. Rules / Invariants
8. Checklist
9. Risks
10. Debug / Common Failure
11. Prompt Template
12. Version / Owner / Evidence

## Golden Rule
Do not store knowledge only as a long article. Every useful knowledge asset must include a checklist and reusable prompt.

## File Naming
Use numeric prefixes for search and ordering:
- 000_MASTER_SKILL_ROUTER.md
- 010_RTAFNC_OBE_SKILL.md
- 020_ASSESSMENT_SYSTEM_SKILL.md
- 030_THAI_DOCX_SKILL.md
