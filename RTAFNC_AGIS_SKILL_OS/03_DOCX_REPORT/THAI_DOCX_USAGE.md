# Thai DOCX Usage — RTAFNC

## Use When
- OBE/SAR book
- Official report
- Thai memo
- Thai academic or administrative document

## Rules
- Use TH Sarabun New or TH SarabunPSK.
- Check Thai line breaking.
- Check tables do not overflow pages.
- Check headings, page numbers, and table of contents.
- For official documents, check emblem, Thai numbering, margins, and closing wording.
