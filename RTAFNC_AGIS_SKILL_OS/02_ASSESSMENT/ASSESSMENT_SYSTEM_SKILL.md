# Assessment System Skill — RTAFNC

## Mission
Create assessment systems from real data structures, not guessed headers or guessed scoring rules. Produce traceable xbar/S.D. reports, dashboard, HTML/PDF/Excel outputs, and OBE evidence.

## Core Rules
1. Inspect real sample files before designing.
2. Separate raw data from output reports.
3. Do not hardcode number of questions or students when headers can be read.
4. Formulas must be inspectable and traceable.
5. Reports must include overall summary, item-level summary, comments, and evidence linkage.

## Use With OBE
- Link course scores to CLO.
- Link evaluation items to Sub-PLO.
- Use results as CQI input.
