# CQI Loop Template — OBE/SAR RTAFNC

## Purpose
Show that assessment results are not only collected, but are used to improve curriculum, teaching, assessment, and student outcomes.

## CQI Fields
| Field | Description |
|---|---|
| Cycle | improvement cycle code such as CQI-01 |
| Data Source | source such as Military Training 1 result, simulation log, graduate feedback |
| Finding | what was found |
| Root Cause | why it happened |
| Improvement Action | what will be changed |
| Owner | responsible person/unit |
| Timeline | target date |
| Evidence After Action | evidence after improvement |
| Result | effect after improvement |

## Example
Finding: Military Training 2-3 have evaluation forms but CLO mapping is not explicit.
Root Cause: No common CLO/rubric standard for years 2-4.
Action: Build CLO Matrix and rubric for Military Training 2-4.
Evidence: CLO/Rubric V1 and curriculum committee minutes.
