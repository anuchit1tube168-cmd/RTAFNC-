# PLO/Sub-PLO Dictionary — RTAFNC

## Recommended Action Verbs
| Verb | Use when | Example |
|---|---|---|
| ปฏิบัติ | performance must be observed | ปฏิบัติตามแบบธรรมเนียมทหารได้ |
| แสดงทักษะ | performance must be demonstrated | แสดงทักษะบุคคลท่ามือเปล่าได้ถูกต้อง |
| บูรณาการ | multiple domains must be connected | บูรณาการพยาบาล ทหาร และการส่งต่อทางการแพทย์ |
| ประยุกต์ใช้ | knowledge is applied to a situation | ประยุกต์ใช้หลักฐานเชิงประจักษ์ในการดูแลผู้รับบริการ |
| ประสานงาน | work involves multiple parties | ประสานทีมสุขภาพและหน่วยทหารได้ |
| ตัดสินใจ | decision under conditions/risk is assessed | ตัดสินใจในสถานการณ์จำลองได้อย่างปลอดภัย |

## Avoid in PLO/Sub-PLO
- เข้าใจ
- เรียนรู้
- ได้รับการฝึก
- มีความรู้
- มีวินัย without observable behavior
- Command or tactical warfare terms when not truly assessed at that level

## Sub-PLO Formula
Action Verb + Context + Standard + Evidence

Example:
แสดงภาวะผู้นำ ผู้ตาม การทำงานเป็นทีม การแก้ปัญหา และการประสานงานในกิจกรรมการฝึกและสถานการณ์จำลองทางทหารได้เหมาะสมกับบทบาทนักเรียนพยาบาลทหารอากาศ
