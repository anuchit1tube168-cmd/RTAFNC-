# RTAFNC AGIS Skill OS

ชุดนี้คือระบบจัดการ Skill สำหรับงาน วพอ.พอ. โดยเอา Skill ที่มีอยู่ใน Google Drive และในแชทมาจัดเป็นระบบใช้งานจริง ไม่ใช่แค่เก็บไฟล์ไว้เฉย ๆ

## หลักการ
1. Fable Workflow เป็นสมองกลาง: INSPECT → CLARIFY → DESIGN → BUILD → VERIFY → DELIVER
2. Professional Skill MD Standard เป็นมาตรฐานการจัดความรู้
3. ทุกโปรเจกต์ต้องมี SKILL.md และ WIKI.md
4. ทุกงาน OBE ต้องมี Mapping: PLO → Sub-PLO → CLO → Assessment → Evidence → CQI
5. งานเอกสารไทยให้ใช้ thai-docx เพื่อกันปัญหาตัดบรรทัดและ layout เพี้ยน

## โครงสร้าง
- 00_MASTER: ตัว Router และกติกากลาง
- 01_OBE: Skill เฉพาะ OBE/Sub-PLO/ทหาร 1–4/SOS150
- 02_ASSESSMENT: แบบประเมินและรายงาน x̄/S.D./Dashboard
- 03_DOCX_REPORT: เอกสาร Word ภาษาไทย
- 04_WEBAPP: ระบบเว็บ/Good Deeds/LIFF/Dashboard
- 05_VIDEO_TO_SKILL: ถอดคลิปเป็น Skill MD
- 06_GRAPHIFY: วิเคราะห์ระบบหลายไฟล์เป็น Knowledge Graph
- 99_EXPORTS: ไฟล์ Excel และ preview ที่สร้างแล้ว

## วิธีใช้เร็ว
เปิด `00_MASTER/MASTER_SKILL_ROUTER.md` ก่อนทุกครั้ง แล้วดูว่างานเข้ากลุ่มไหน จากนั้นค่อยใช้ Skill เฉพาะโดเมน
