# RTAFNC AGIS Skill OS Index

This page is the quick entry point for reusable skills used in RTAFNC education, discipline, OBE, assessment, documents, web apps, and knowledge extraction work.

## Start Here
1. Open `00_MASTER/MASTER_SKILL_ROUTER.md`.
2. Identify the work type.
3. Use the matching skill folder.
4. Verify outputs before delivery.

## Safety Rule
Only public-safe markdown files should be committed to this public repository. Internal evidence files, spreadsheets, PDFs, student data, and official documents should stay in Drive or private storage unless cleared.
